ABOUT ME

My story

ผมชื่อ นายศุภวิชญ์ ร่องพืช หรือ เบส เกิดเมื่อวันที่ 4 ตุลาคม พ.ศ. 2550 ปัจจุบันอาศัยอยู่ที่บ้านเลขที่ 275/2 ตำบลหนองม่วงไข่ อำเภอหนองม่วงไข่ จังหวัดแพร่ ผมจบการศึกษาระดับประถมศึกษาจากโรงเรียนมารดาอุปถัมภ์ และจบมัธยมศึกษาตอนต้นจากโรง เรียนพิริยาลัย จังหวัดแพร่ ขณะนี้กำลังศึกษาในระดับประกาศนียบัตรวิชาชีพชั้นปีที่ 2 ที่วิทยาลัยเทคนิคแพร่ ผมสนใจศึกษาเกี่ยวกับการเขียนโปรแกรมและการลงทุนอย่างมาก โดยมุ่งเน้นพัฒนาทักษะในด้านนี้อยู่ตลอดเวลา

ในอนาคต ผมมีความฝันที่จะเป็น Full-Stack Developer ผู้เชี่ยวชาญในการพัฒนาเว็บไซต์และแอปพลิเคชัน ทั้งในส่วนหน้าบ้าน และหลังบ้าน แรงบันดาลใจมาจากความหลงใหลในงานเขียนโค้ดและการจัดการระบบ นอกจากการเรียนแล้ว ผมยังให้ความสำคัญกับ การศึกษาเรื่องการลงทุน เพื่อเตรียมความพร้อมสำหรับการสร้างความมั่นคงทางการเงินในอนาคต

My Skill

ผมมีความถนัดในการเขียนโปรแกรมภาษาต่างๆ ได้แก่ HTML, CSS และ JavaScript ซึ่งสามารถสร้างและพัฒนาเว็บไซต์ที่ตอบโจทย์การใช้งานได้อย่างดีเยี่ยม มีทักษะในการออกแบบหน้าเว็บด้วย CSS ที่เน้นความสวยงามและการใช้งานที่ง่าย ควบคู่ไปกับการใช้ JavaScript เพื่อเพิ่มฟังก์ชันการทำงานแบบโต้ตอบ มุ่งมั่นที่จะพัฒนาทักษะใหม่ๆ อย่างต่อเนื่องเพื่อให้ทันต่อการเปลี่ยนแปลงของเทคโนโลยีและอุตสาหกรรม
